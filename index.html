üß† What is Executive Function?

The mental skills that help you: start tasks, stay focused, remember instructions, manage time, and switch between activities. Many kids with dysgraphia/dyspraxia also have executive function challenges.

‚è∞ Time Management Strategies

Pomodoro Technique: Work 25 minutes ‚Üí 5 minute break ‚Üí Repeat

Visual timers: Time Timer app‚Üó shows time "disappearing"

Backward planning: Start with due date, work backward to figure out when to start

üìã Breaking Down Big Tasks

Instead of: "Write book report" (too big, causes paralysis)

Break into:
1. Read chapters 1-3 (30 min)
2. Write 3 key points about characters (10 min)
3. Dictate intro paragraph (5 min)
4. Dictate body paragraphs (15 min)
5. Write conclusion (5 min)
6. Review and edit (10 min)

üéØ Focus Tools

üóÇÔ∏è Organization Systems

Physical space: Designate ONE spot for backpack, laptop, chargers ("launch pad")

Digital space: Folder system: School ‚Üí [Subject] ‚Üí [Type: Notes, Homework, Tests]

Daily routine: Same time, same place for homework every day = less mental energy

üé≤ Gamification Strategies

Turn executive function into a game:

‚Ä¢ Todoist karma points for completing tasks
‚Ä¢ Forest app growing virtual forest
‚Ä¢ Personal challenge: "Can I start homework within 5 minutes today?"
‚Ä¢ Reward system: Points toward chosen activity/treat

üí° Key insight: You're not lazy! Your brain needs external systems (apps, timers, alarms, lists) to do what others do internally. That's completely normal for dysgraphia/dyspraxia/ADHD.
